# Bug Report: Form Allows Empty Submission

**Severity:** High  
**Environment:** Chrome (Mac), Web Application  

## Steps to Reproduce
1. Go to the sign-in page  
2. Leave both First Name and Last Name empty  
3. Click "Sign In"  

## Expected Result
System should prevent submission and display an error message.

## Actual Result
Form allows submission without validation.

## Notes
This can lead to invalid or incomplete client records in the system.
