# Exploratory Testing: Technical Assessment

## Context
Performed exploratory testing on a web application as part of a timed assessment.

## Approach
- Explored user and admin workflows  
- Tested edge cases and UI consistency  
- Focused on usability and interaction flow  

## Findings
- UI inconsistencies across pages  
- Missing validation in forms  
- Workflow confusion in admin panel  

## Outcome
Identified multiple issues within a limited timeframe and documented findings clearly.
