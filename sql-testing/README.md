# SQL Data Validation Testing – Northwind

## Objective
To validate data integrity, consistency, and business logic in the Northwind sample database using SQL queries.

## Database
- MySQL Workbench
- Northwind sample database

## Dataset

This project uses the Northwind sample database.

Source:
https://github.com/dalers/mywind

The dataset was imported into MySQL Workbench for testing purposes.

## Testing Focus
- Missing or null values
- Duplicate records
- Foreign key / relationship consistency
- Business rule validation
- Aggregate and summary checks

## Skills Demonstrated
- SQL querying
- Data validation testing
- Database integrity analysis
- QA-style anomaly detection
- Business logic verification
