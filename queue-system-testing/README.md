# Queue Management System – QA Testing

## Objective
This system was a personal project I built to simulate a real-world queue management workflow, focusing on improving client tracking and operational flow.

For testing, I validated data integrity and backend behavior by evaluating UI interactions, data storage, and overall system responses.

Note: The application was tested primarily in Google Chrome. For best results, please open the links by copy-pasting them into the browser.
